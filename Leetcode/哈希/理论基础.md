考虑数组/set/map作为哈希表。

规模小：数组
规模大：set（要求有顺序时）/map（要求key-val时）

# unordered_set

| 方法名               | 功能说明                                                               |
| -------------------- | ------------------------------------------------------------------ |
| `insert(value)`      | 插入                                          |
| `erase(value)`       | 删除                                                          |
| `find(value)`        | 查找，返回一个迭代器，指向val，如果找不到，则返回s.end()，即 尾后迭代器        |
| `count(value)`       | 返回值的出现次数（`set` 中是 `0` 或 `1`）                                       |
| `size()`             | 返回当前元素个数                                                           |
| `empty()`            | 判断集合是否为空                                                           |
| `clear()`            | 清空所有元素                                                             |

遍历：
```c++
for (int x : s) {
    cout << x << " ";  
}
```
